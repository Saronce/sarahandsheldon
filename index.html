Text Editor

So for the actual "content" that we are writing it would be nice to have most, or all, of the pictures we upload actually in with the text. You were probably thinking that was how we were going to do it any ways. The problem is, though, that we can't use Microsoft Word or even Open Office. The reason for this is kinda technical but basically it will make it impossible to convert the files into a website or to read them with anything but Word! The best option would be HTML since that is what websites are written in anyways and it can be read by pretty much anything. Unfortunately it might be a pain in the ass for you to learn, although it is not nearly as complicated as a programming language. There is also Markdown. It works, on the technical level, exactly like HTML but it is way easier to use (its what I'm using right now to write this). Its still not really like Word at all but its very simple and if you wanted to you could learn how to use it in minutes. Basic "Plain Text"/TXT/.txt files are also good because they can easily be put into a website and can be read by pretty much any text-editor. However there is the huge drawback of not being able to add pictures and you can't really make it look pretty, AT ALL!

So ya, I guess I think Markdown is the best option. I'm not really sure what other options there are. If you have any ideas let me know!

To use Markdown download MarkdownPad (http://markdownpad.com/) and install it. The screen on the right is what the document will look like on a website, the left is the part you are actually editing. All you really need to learn is things like "**" which makes a section of text bold, etc. Pretty much just write as you normally would until you have a problem or want to do something specific (like add a picture, make columns, or make a heading, etc) then just google how to do it or go to "Help">"Official Markdown Syntax Guide" (in MarkdownPad). I can help you out with Markdown if you want, call me. Also I am not just wanting to use Markdown so that I can be in a position of power or something like that... I would prefer to use Word too... its just that I would also really like to be able to do lots of stuff with the files we end up writing. Also Word files have the issue of going out of date (which HTML doesn't really) and I want to be able to read these files when I'm 80 in my holographic room on Mars!!!

Adding Pictures, and an Example

So, assuming you download MarkdownPad, here is how we integrate pictures with our writing:

So to add pictures in HTML/Markdown you basically just make a link to the URL of the picture you want and then it shows up on your page. The problem is that the picture you want is probably just on your hard-drive and doesn't have a URL. So to solve this problem we have to use an internet picture hosting service... Ian recommended Flickr, cause it's free.

So if you don't have a Flickr account make one, and then add me as a friend/contact so that we can see and use each other's pictures. Then upload the pictures you want to use. Then go to the specific picture you are trying to add and click on the drop-down menu above it which is labelled "share". Then select the "Grab the HTML/BBCode" heading. Select the size you want the picture to be, and make sure that "HTML" is selected not "BBCode". Then copy the text in the box (cntrl-c). Then go to your Markdown text document and paste the text you just copied wherever you want the picture to appear. Thats it! The picture should then appear in the right screen in MarkdownPad (press f6 to see your document in your web browser).
